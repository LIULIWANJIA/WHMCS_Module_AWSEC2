## 使用方法
 - 把AWSEC2目录直接扔到 WHMCS/modules/servers 下即可
 - 自定义字段 cloudinit (文本框 textarea 在订单页面显示) pem (文本框 textarea 仅管理员可见) data (文本框 textarea 仅管理员可见)
## 特性
- 动态IP (关机再开机即可更换IP)
- 支持AGA (貌似没啥用)
- 支持全区，多账号
